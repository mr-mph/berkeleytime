# Units 1–3 standard double reconstruction

This room is the shared standard high-rise double example used by UC Berkeley
Housing for Units 1, 2, and 3. Berkeley publishes the same double-room
description for all three Units and cautions that individual rooms vary in
size, layout, and furniture configuration.

## Source-backed envelope and arrangement

- Typical footprint: 13′9″ × 13′2″ (4.19 × 4.01 m), from the published UC
  Berkeley college-information sheet used by the existing Unit triple spec.
- Entry wall: centered 0.91 m inward-swinging door, with one open closet,
  mirror, and light for each resident on opposite sides.
- Opposing wall: centered, three-panel window with radiator below.
- Side walls: one maple Twin XL bed and one tall bookshelf per side. The beds
  use under-bed drawer banks visible in the supplied room photographs.
- Center: two desks placed back-to-back so one chair approaches from each
  resident's side. The MicroChill sits at the window end of the center aisle.
- Room shell and standard Unit furniture use the same finish family as the
  Units 1–3 triple reconstruction.

## Coordinate system

- Origin: inner-left corner of the entry wall, at the floor.
- +x: left to right while standing in the doorway.
- +z: doorway to window.
- +y: up.

The source of truth for exact constants is `src/unit-double/spec.js`.
